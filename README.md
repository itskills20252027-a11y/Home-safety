Purpose:
The main purpose of the home safety management is to protect residential houses from security threats & alert the houseowner with notification

Scope:
The scope of this project is to prevent houses from danger & alert the user with the notification to inform them

Functional Requirements
The system shall allow users to login with username & password
System shall continoulsy monitor windows & door
System shall detect gas leakage
System shall detect smoke using sensors
System shall trigger alarm if unauthorized access has been identified
System shall send notifications to the owner

Non-Functional Requirements

The system shall response to every action
The system should detect multiple sensors at a single time
The system should keep the password encrypted
The system should be available 99% of the time'
The system should have easy user interface
